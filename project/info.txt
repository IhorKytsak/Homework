Додаток містить 3 сторінки:
    •	Головна сторінка 
    •	Сторінка додавання нового сету;
    •	Змінення вибраного сета.

Головна сторінка показує список наборів та має наступні можливості:
    • Додати новий набір .Після натискання кнопки "Add new" система перенаправляє на сторінку "Додати новий сет"
Під кожним набором є дві кнопки:
    • Edit (відкриває зміну набір сторінок);
    • Remove (видаляє набір зі списку).
Після після позначення набору він відмічається як вивчене та перемщається до кінця списку.

 Сторінка "Додати новий сет"
На сторінці "Додати новий елемент" відображаються порожні поля введення 
"Ім'я", "Додати терміни", "Зберегти зміни" та "Скасувати".
Набір можна зберегти, лише якщо "Ім'я" воно заповнене.
Кнопка "Додати терміни" повинна додати блок із двома входами та кнопку "Видалити". 

На сторінці "Змінити сет" відображаються поля введення з відповідним значенням, кнопки "Зберегти зміни" та
 "Скасувати". Змінений сет можна зберегти, коли ім’я не порожнє. Після успішної модифікації додаток 
 перенаправляє на головну сторінку. 
Кнопка "Скасувати" перенаправляє користувача на головну сторінку без збереження.


Особливості:
1. Після оновлення сторінки зберігаються всі раніше додані елементи.(localStorage для зберігання даних).
2. Усі сторінки додаються в <div id = "root"> </div>, який знаходиться всередині тегу body засобами Java Script.
